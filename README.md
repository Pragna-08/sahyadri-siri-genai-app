# Sahyadri-Siri GenAI App

## 📖 Overview
Sahyadri-Siri is an Android application developed using Generative AI technologies to provide intelligent assistance and interactive user experiences. The application leverages AI capabilities to understand user queries and generate relevant responses in real time.

## ✨ Features
- AI-powered conversational interface
- Intelligent response generation
- User-friendly Android application
- Real-time interaction and assistance
- Modern and responsive UI
- Integration with Generative AI models

## 🛠️ Technologies Used
- Android Studio
- Java / Kotlin
- Generative AI APIs
- XML for UI Design
- Git & GitHub

## 📂 Project Structure

```text
app/
├── src/
├── res/
├── manifests/
└── build.gradle
```

## 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/Pragna-08/sahyadri-siri-genai-app.git
```

2. Open the project in Android Studio.
3. Sync Gradle files.
4. Build and run the application on an Android device or emulator.

## 📱 Usage

- Launch the application.
- Enter your query in the chat interface.
- Receive AI-generated responses instantly.
